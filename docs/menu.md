# Меню

[← README](../README.md)


Открыть интерактивное меню:

```bash
xray-menu
```

В шапке меню выводятся название `Xray VPS Manager`, установленная версия Xray, версия менеджера, дата последнего обновления менеджера в выбранном часовом поясе, время обновления `geoip.dat`/`geosite.dat` и время последней проверки безопасности:

```text
+--------------------------------------------------------------------------------+
| Xray VPS Manager                                                               |
+-----------------+--------------------------------------------------------------+
| Xray Version    | 26.3.27                                                      |
| Manager Version | v1.0.0                                                       |
| Manager Updated | 2026-06-12 10:34 MSK                                         |
| Geo Assets      | geoip.dat: 2026-06-11 13:26 MSK; geosite.dat: 2026-06-11...  |
| Security Audit  | 2026-06-05 15:55 MSK                                         |
+-----------------+--------------------------------------------------------------+
```

Если проверка безопасности ещё не выполнялась или была больше 30 дней назад, под шапкой выводится рекомендация запустить `Безопасность` -> `Проверить безопасность сервера`.

Пункт `Настроить оплату и округление` показывает текущую payment-сводку, затем позволяет изменить месячную аренду сервера, годовую аренду домена и округление суммы на платного клиента. Годовая аренда домена делится на 12 и прибавляется к месячной аренде сервера перед делением между платными клиентами.

После выбора команды её вывод отделяется визуальным блоком:

```text
==================== Показать клиентов =====================

... вывод команды ...

============================================================
```

Через меню можно менять `PORT`, `REALITY_SNI`, `FINGERPRINT`, создавать VLESS Reality/TLS и Trojan TLS/WebSocket подключения через Caddy, управлять клиентами, сроком доступа клиентов, каскадом, стартовой ссылкой, обновлением и откатом Xray.
В Telegram-админке `/admin` раздел `Клиенты` содержит действия `Добавить клиента`, `Получить VLESS-ссылку`, `Подписки клиентов` и `Продлить подписку`; получение ссылки показывает список клиентов и отправляет актуальный ключ в блоке кода для пересылки.
При обновлении `PORT` или `REALITY_SNI` значение `REALITY_DEST` пересчитывается автоматически как `REALITY_SNI:443`.
Публичный `PORT` - это порт входа клиентов в Xray. `REALITY_DEST` - отдельная цель маскировки, она не должна повторять нестандартный входной порт вроде `8443` или `2053`.
После смены `FINGERPRINT` перезапуск Xray не нужен, но клиентам нужно заново выдать ссылку.
Торренты по умолчанию запрещены. Управлять доступом можно через `Маршрутизация` -> `Торренты`.
Часовой пояс можно посмотреть или изменить через `Сервис и диагностика` -> `Показать часовой пояс` или `Изменить часовой пояс`.
В `Трафик и активность` -> `Просмотр трафика` сначала показывается суммарный расход всех клиентов за текущий месяц и список клиентов с месячным трафиком. После выбора клиента можно посмотреть день по часам, неделю по дням, месяц по дням или произвольный период. Для клиентов с несколькими credentials эти отчёты дополнительно показывают блок `Credentials` с расходом по каждому protocol credential.
В `Трафик и активность` -> `Настройки суммарного трафика` можно включить строку `TOTAL x2` и изменить множитель для оценки расхода VPS-провайдера, который считает входящий и исходящий трафик вместе.
В `Трафик и активность` можно посмотреть activity-отчёт по клиенту, подозрительную активность, экспортировать метаданные и управлять архивами экспорта. Клиентский activity-отчёт для multi-credential клиента показывает отдельный блок `Credentials` по `connection`.
В `Трафик и активность` -> `Настройки журнала активности` можно включить или отключить только парсер access log, синхронизировать события, изменить срок хранения детальных событий и настроить пороги suspicious.
В `Маршрутизация` -> `GeoIP routing` можно отдельно включить GeoIP routing-предупреждение для проверки split tunneling или отключить его.
В `Маршрутизация` -> `Блокировки IP/доменов` можно добавить IP или домен из GeoIP RU-событий выбранного клиента, указать комментарий и срок блокировки, посмотреть последнюю дату срабатывания и статистику по клиентам. Сама блокировка глобальная и применяется ко всем клиентам.
В `Клиенты` -> `Лимиты трафика` можно задать лимит трафика в гигабайтах на день или месяц. По умолчанию у клиента нет лимита.
В подменю `Клиенты` -> `Добавить клиента` можно ввести `ИМЯ` или `ИМЯ ДНЕЙ`, например `data_test2 1`. Если такое имя уже существует, меню отменит действие и предложит использовать `Клиенты` -> `Добавить подключение к клиенту`.
Если подключений несколько, меню предложит выбрать подключение из таблицы. В этой таблице доступны и VLESS, и Trojan подключения, поэтому нового клиента можно сразу добавить в нужный протокол.
В `Клиенты` -> `Добавить подключение к клиенту` сначала выбирается существующий клиент из таблицы, затем меню показывает только connections, которых у клиента ещё нет. Команда не меняет срок доступа или статус оплаты клиента.
В остальных действиях подменю `Клиенты` клиент выбирается из таблицы по номеру:

```text
+---+------------+------------+----------+------------------+----------------------+
| № | CONNECTION | NAME       | STATUS   | ACCESS UNTIL     | CREATED              |
+---+------------+------------+----------+------------------+----------------------+
| 1 | default    | starter    | enabled  | бессрочно        | 2026-06-03T22:37:11Z |
| 2 | backup443  | phone      | enabled  | 2026-07-04 00:00 | 2026-06-03T22:38:51Z |
| 0 | Назад      |            |          |                  |                      |
+---+------------+------------+----------+------------------+----------------------+
```
